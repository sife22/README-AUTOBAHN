1 - Objectif

L'idée principale d'Autobahn est de faciliter la location de voitures au Maroc, en rapprochant le client de sa voiture préférée :

  • Il a la possibilité de consulter l'ensemble des véhicules disponibles.
  • Il a la possibilité de réserver les véhicules disponibles.
  • Il a la possibilité d'annuler sa réservation.

2 - introduction

Quelles stratégies, actions et solutions mettre en place pour assurer la croissance d'une plateforme de location de voitures, et augmenter l'expèrience utilisateur dans un marché concurrentiel ?


3 - Concept de base

↩

  • Agence :
    • Présenter les annonces de location en indiquant les informations essentielles du voiture.
    • Modifier les annonces.
    • Supprimer les annonces.
  • Client :
    • Consulter toutes les annonces.
    • Prendre contact avec l'agence de voiture concernée.

4 - Pourquoi pas

Si un espace est disponible pour présenter les offres de vente, cela serait un atout supplémentaire pour l'écosystème de la plateforme.

  • Agence :
    • Publier les annonces de vente en indiquant son numéro de téléphone.
    • Modifier les annonces.
    • Supprimer les annonces.
  • Client :
    • Consulter toutes les annonces.
    • Prendre contact avec l'agence de location de voiture concernée.
